# Not a Mac Clone

GTK theme collection for Linux desktops - gnome-shell, gtk-2.0, gtk-3.0, and gtk-4.0 themes in one package.

## Installation

Copy the theme folders to your system theme directory:
- `/usr/share/themes/` (system-wide)
- `~/.themes/` (user only)

Or use a theme manager extension.

## Themes

- **gnome-shell** - GNOME Shell theme
- **gtk-2.0** - GTK2 theme
- **gtk-3.0** - GTK3 theme
- **gtk-4.0** - GTK4 theme

Each version includes both light and dark variants.
